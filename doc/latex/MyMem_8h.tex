\subsection{Description détaillée}
Memory management functions (user interface). Two versions of memory allocation/deallocation are available. If compiled with M\-E\-M\-T\-R\-A\-C\-E defined, C\-Unit keeps track of all system allocations \& deallocations. The memory record can then be reported using C\-U\-\_\-\-C\-R\-E\-A\-T\-E\-\_\-\-M\-E\-M\-O\-R\-Y\-\_\-\-R\-E\-P\-O\-R\-T. Otherwise, standard system memory allocation is used without tracing.
