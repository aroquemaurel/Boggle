Runs C\-Unit tests using the automated interface. 

This function sets appropriate callback functions, initializes the test output files, and calls the appropriate functions to list the tests and run them. If an output file name root has not been specified using \hyperlink{group__Automated_ga60583b154146b787f03267fc320f86e5}{C\-U\-\_\-set\-\_\-output\-\_\-filename()}, a generic root will be applied. Generates an xml file containing a list of all tests in all suites in the active registry. 

The output file will be named according to the most recent call to \hyperlink{group__Automated_ga60583b154146b787f03267fc320f86e5}{C\-U\-\_\-set\-\_\-output\-\_\-filename()}, or a default if not previously set.

Sets the root file name for automated test output files. 

The strings \char`\"{}-\/\-Listing.\-xml\char`\"{} and \char`\"{}-\/\-Results.\-xml\char`\"{} are appended to the specified root to generate the filenames. If sz\-Filename\-Root is empty, the default root (\char`\"{}\-C\-Unit\-Automated\char`\"{}) is used.
